# Podcast Episodes

This folder contains references to podcast episodes featuring motivational speakers and thought leaders included in this repository. These episodes provide long-form discussions around leadership, teamwork, culture, resilience, and performance.

Podcast entries are documented for educational and reference purposes only.

---

## What Belongs Here

Each file in this folder should represent a single podcast episode and may include:

- Episode title  
- Podcast name and host  
- Brief summary of discussion topics  
- Key themes or takeaways  
- Link to the original episode (audio or video)  

---

## File Naming Guidelines

- Use lowercase letters  
- Use hyphens instead of spaces  
- Keep filenames descriptive  

**Examples:**
- `podcast-leading-through-adversity.md`
- `podcast-building-strong-teams.md`

---

## Contribution Guidelines

- Only include publicly available podcast episodes  
- Clearly attribute the podcast, host, and platform  
- Keep summaries factual and neutral  
- Avoid promotional or sales-oriented language  
- Maintain consistent formatting across all files  

---

## Related Folders

This folder complements other content types in the speaker’s section, including:

Together, these folders provide a broader view of each speaker’s ideas and impact.
